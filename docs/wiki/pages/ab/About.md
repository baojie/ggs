# 关于本 Wiki

## 语料来源

本 Wiki 基于贾雷德·戴蒙德（Jared Diamond）所著《枪炮、病菌与钢铁：人类社会的命运》（*Guns, Germs, and Steel: The Fates of Human Societies*，1997）谢延光中译本构建。全书共 21 章（含前言、后记），分为 4 个部分。

- **作者**：贾雷德·戴蒙德
- **原著**：*Guns, Germs, and Steel*（1997 年，Pulitzer Prize 获奖作品）
- **译者**：谢延光
- **章节**：前言 + 19章 + 后记，共分四部分

## 结构化知识

本 Wiki 将全书内容分解为词条，每个词条通过 PN（段落编号）系统关联到书中具体段落，实现精确引用与跨章关联。

PN 编号格式为 `[NNN-PPP]`，NNN 为章号，PPP 为段序号。例如 `[008-015]` 表示第八章第 15 段。

## 页面类型

本 Wiki 词条分为以下类型：

| 类型 | 说明 |
|------|------|
| concept | 核心概念与理论 |
| person | 人物 |
| event | 历史事件 |
| place | 地理区域 |
| species | 物种 |
| organization | 机构 |
| chapter | 章节摘要 |
| overview | 综述/目录页 |
| list | 列表页 |

## 使用说明

- 搜索框支持词条名、别名全文检索
- 每个词条页面标注「相关章节」，可追溯原文位置
- 顶部导航「所有页面」可浏览全部词条
