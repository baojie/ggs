## 简介

引用次数（total_refs）反映一个词条被其他页面通过 wikilink 引用的频率，是衡量概念在全书论证网络中核心程度的重要指标。本页面列出引用次数最高的词条排行。

## 最高引用 Top 50

::: query
sort: total_refs
order: desc
limit: 50
display: table
fields: [label, type, quality, total_refs]
field_labels:
  label: 词条
  type: 类型
  quality: 质量
  total_refs: 引用数
title: 被引用最多 Top 50
:::

## 说明

- 引用次数基于所有 wiki 页面中的 wikilink 统计，不含章节页和索引页自身的引用。
- 词条的引用次数随新建页面和 wikilink 注入而持续增长，本页面动态更新。
