## 简介

本页按质量等级列出 wiki 中的所有页面。质量等级反映内容的充实程度：featured 为完整条目，standard 为中等条目，basic 为基础条目。质量分布是 GROW 阶段推进的核心指标——stub 比例和 featured 比例共同决定下一阶段的优化方向。

## Featured 条目

::: query
quality: featured
sort: label
order: asc
display: table
fields: [label, type, description]
field_labels:
  label: 页面
  type: 类型
  description: 描述
title: Featured 条目
:::
